Transformation formulas

These transformations are also known, in GNSS world, as Helmert transformations or seven parameters transformations. Seven parameters are needed: three rotation angles, three coordinates of the translation and a scale factor. The rotation angles and the scale factor are treated as infinitesimals, so the implemented formula must be considered right at the first order. The transformation, according to ITRF, and also EUREF, reads: 

      | X_end |        |  1 + k   -R_Z    R_Y   | | X_start |       | T_X |
      |       |        |                        | |         |       |     |
      | Y_end |   =    |   R_Z    1 + k  -R_X   | | Y_start |   +   | T_Y |
      |       |        |                        | |         |       |     |
      | Z_end |        |  -R_Y     R_X   1 + k  | | Z_start |       | T_Z |
where:
  • k is the scale factor;
  • R_X(Y,Z) is the rotation angle around X(Y,Z)-axis;
  • T_X(Y,Z) is the traslation along the X(Y,Z)-axis
Velocity on cartesian coordinates V are needed to transform the parameter P from the starting epoch t_1 to the final epoch t_2 according to the formula: 
      P(t_2) = P(t_1) + V * (t_2 - t_1)
where P can be X, Y or Z and V the corresponding velocity.
